Limit the number of posts that your editors and authors can write.

== Description ==

Limit the number of posts that your editors and authors can write.  This is great for blog sites that have tons of editors and authors but dont want them to post too many posts.  This plugin gives the admin the capabilites to regualate how many posts are being published by the authors and editors.

== Installation ==

1. Upload `limit-posts` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
